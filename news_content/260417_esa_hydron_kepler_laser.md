# Proyecto HydRON: La ESA y Kepler construirán una red de fibra óptica láser celestial

**Fecha:** 2026-04-17

Para superar las limitaciones de radiofrecuencia, la ESA adjudicó un contrato a Kepler Communications para la fase 3 del proyecto HydRON. Esta iniciativa pretende establecer enlaces ópticos con velocidades de terabits por segundo, creando un internet láser espacial.
